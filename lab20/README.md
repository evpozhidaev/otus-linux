### Диаграмма
```
                                                +----------+
                                                | Internet |
                                                +----+-----+
                                                     |
                                                     |
                                                     |
                                               +---+-+------+
                                               | inetRouter |
                                               |            |
                                               +----+--+----+
                                                    |  |.1
                                           team0    |  |
                                        (eth1,eth2) |  | 192.168.255.0/30
                                        +---------->|  |   [router-net]
                                                    |  |
                                                    |  |.2
                                             +------+--+-----+
                                             |               |
                                             | centralRouter |
                                             |               |
                                             |               |
                                             +--+---------+--+
                                        eth3.100|         |eth3.101
                                        (w/o IP)|         |(w/o IP)
                                                |         |
                                                |         |
                                                |         |
                                                |         |
                                                |         |
      +-------------+ 10.10.10.254/24 +---------++       ++---------+ 10.10.10.254/24 +-------------+
      | testServer1 +-----------------+          |-------|          +-----------------+ testServer2 |
      +-------------+                 | VLAN 100 |       | VLAN 101 |                 +-------------+
      +-------------+ 10.10.10.1/24   |          |       |          | 10.10.10.1/24   +-------------+
      | testClient1 +-----------------+          |       |          +-----------------+ testClient2 |
      +-------------+                 +----------+       +----------+                 +-------------+
                                            |                  |
                                            |  VIRTUAL SWITCH  |
                                            |                  |
                                            +------------------+
```
### Стенд

Стенд разворачивается через [Vagrantfile](Vagrantfile) в котором при провижиненге последней VM запускается ansible-playbook
